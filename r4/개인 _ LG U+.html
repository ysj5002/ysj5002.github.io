가능한 개인명의 고객만 이용 가능합니다.

간편조회 이용 시 확인 하세요
  1. 1. 간편조회 서비스 이용 가능 조건
    • U+ ID 회원이 아니어도 유플러스 모바일
      가입자라면 가능 합니다.
    • 유플러스 가입자와 휴대폰 사용자가 다른 경우
      가입자 정보로만 조회가능 합니다.
    • 개인명의 고객만 이용 가능 합니다.
      예) 기업/소상공인 고객, 선불폰 고객, 알뜰폰 고객은 이용 불가
  2. 2. 휴대폰 본인인증 유지시간 (5분)
    • 휴대폰 본인인증 후 5분간 사용하지 않으시면 인증번호를 재발급 받으셔야 합니다.
    • 개인정보 보호를 위해 간편조회 이용 후 브라우저를 종료 하시기 바랍니다.
  3. 3. USIM 개통 고객
    • USIM카드를 휴대폰에 장착한 상태에서 문자로 인증번호 수신이 가능합니다.
  4. 4. 인증문자가 오지 않을 경우
    • 아래와 같이 단말기 설정을 변경해주세요.
      - 휴대폰 문자 설정 메뉴 중 스팸 관련 항목의 번호 없는 메시지 차단 선택해제

    할인유형 선택

    휴대폰 가입시 고객님의 선택에 따라 단말 지원금 할인 또는 요금 할인을 받으실 수 있습니다.

    • 공시지원금(휴대폰 가격 할인) : 휴대폰을 구입하실 때 제공받는 지원금 할인 프로그램 입니다.
    • 선택 약정 할인(통신요금 25% 할인) : 휴대폰 지원금을 받지 않는 경우 지원금 대신 요금 할인을 제공해 드리는 프로그램입니다.
    • 데이터 2배 무약정 프로그램 : 요금제에 따라 약정 요금제에서 제공되는 데이터 월 제공량의 최대 2배 데이터가 제공되는 프로그램입니다.
